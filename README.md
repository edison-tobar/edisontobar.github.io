# Portafolio — Edison Javier Tobar Roblero

Sitio público (GitHub Pages) con portafolio profesional, versión avanzada con dark mode, animaciones GSAP, diseño responsive y subpáginas para cada proyecto.

## Archivos principales
- `index.html` — Página principal (responsive, dark mode, GSAP animations)
- `project-1.html` … `project-4.html` — Subpáginas detalladas de proyectos
- `CV_Edison_Tobar_2025.pdf` — Hoja de vida (asegúrate de subir este archivo)
- `foto_edison_tobar.png` — Foto profesional (asegúrate de subir)

## Publicación en GitHub Pages
1. Asegúrate que el repositorio esté público.
2. Sube los archivos al repo (raíz).
3. En Settings → Pages selecciona `main` y `/ (root)`.
4. URL de publicación: `https://<tu-usuario>.github.io/<tu-repo>/`

---

## Notas
- GSAP se usa desde CDN para animaciones. No requiere compilación.
- El modo oscuro guarda la preferencia en `localStorage`.
- Las subpáginas tienen enlaces de vuelta a `index.html`.
